# Vocabulary Web App

Angular application for vocabulary management.

## Run

From `web-app`:

```bash
npm run serve
```

Default URL: `http://localhost:4200`

## Build

From `web-app`:

```bash
npm run build
```

## Test

From `web-app`:

```bash
npm run test
```

## Proxy / Backend

- Proxy config: `apps/vocabulary/proxy.conf.json`
- Proxy target defaults to `http://localhost:8082`
- Auth and API routes are proxied for local development
